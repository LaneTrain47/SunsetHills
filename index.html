Sunset Hills: A Coding Challenge

This JavaScript coding exercise uses arrays to determine which buildings get the best view of the sunset.
Since the sun sets in the west, the left-most building will always be able to see the sunset regardless of its height. But how about all of the other buildings in the skyline?
